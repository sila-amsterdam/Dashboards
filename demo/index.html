Chatbot Widget

Schaalbare, aanpasbare chatbot widget. Klik op de chat-knop rechtsonder om de demo te starten.

Features

✓ Zero dependencies
✓ Embed via <script> tag
✓ Aanpasbare kleuren & logo
✓ Sessie-geheugen (localStorage)
✓ Multi-tenant (companyId)
✓ NL & EN ondersteuning
✓ Afspraken boeken (Google Calendar)
✓ Mobile responsive
✓ Toegankelijk (ARIA)
✓ Typing indicator

Configuratie opties

webhookUrl

URL van de n8n webhook. Elke POST-aanroep gaat hier naartoe.

companyId

Unieke identifier per bedrijf. Bepaalt welke kennis en sessie geladen wordt.

name

Naam van de assistent, getoond in de header van het chatvenster.

color

Primaire kleur (HEX). Alle kleuren in de widget worden hiervan afgeleid.

logo

URL naar een afbeelding (optioneel). Getoond als avatar in header en berichten.

welcomeMessage

Eerste bericht dat de assistent stuurt wanneer het venster opent.

Kleur varianten

Embed code

Plak dit in de <body> van elke pagina:

<!-- Chatbot configuratie --> <script> window.ChatbotConfig = { webhookUrl: 'https://sila-ai.app.n8n.cloud/webhook/chatbot', companyId: 'company-001', name: 'Uw Assistent', color: '#6366f1', logo: null, // of 'https://uw-site.nl/logo.png' welcomeMessage: 'Hallo! Hoe kan ik u helpen?', language: 'nl', // 'nl' of 'en' position: 'bottom-right' // of 'bottom-left' }; </script> <script src="chatbot-widget.js"></script>
